## Requirements

This project requires the following:

-   [Node.js](https://nodejs.org/en/)

## Workstation setup

Clone the repo to your machine. Then run:

```
npm i
```

## Running the project

Start the server with

```
npm run start
```

Your server should be running at `http://localhost:3001/`
